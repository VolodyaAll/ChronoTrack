# ChronoTrack

Android-приложение для отслеживания времени, потраченного на различные виды активностей. Позволяет записывать, анализировать и визуализировать распределение времени по дням и периодам.

## Возможности

- **Отслеживание активностей** — переключение между активностями одним нажатием, с возможностью задать время начала ретроспективно
- **Комментарии к записям** — текстовые заметки, фото, видео и аудио, привязанные к периодам активности
- **Статистика** — круговые диаграммы, фильтрация по дате и времени, детальная разбивка по дням
- **Фоновый сервис** — непрерывное отслеживание через foreground service с уведомлением
- **Локализация** — русский и английский языки с возможностью переключения в настройках
- **Управление активностями** — создание, редактирование, архивирование, восстановление и удаление

## Стек технологий

| Компонент | Технология |
|---|---|
| Язык | Kotlin 2.0 |
| UI | Jetpack Compose + Material 3 |
| Архитектура | MVVM |
| База данных | Room (SQLite) |
| Асинхронность | Kotlin Coroutines + Flow |
| Навигация | Navigation Compose |
| Графики | MPAndroidChart |
| Изображения | Coil |
| Настройки | DataStore Preferences |

## Структура проекта

```
app/src/main/java/com/sharai/chronotrack/
├── data/
│   ├── dao/            # Room DAO-интерфейсы
│   ├── database/       # AppDatabase, конвертеры типов
│   ├── model/          # Entity-классы (Activity, TimeEntry, Comment)
│   └── preferences/    # DataStore настройки
├── repository/         # Репозитории для доступа к данным
├── service/            # Foreground service для отслеживания
├── ui/
│   ├── comments/       # Диалоги для работы с комментариями
│   ├── components/     # Переиспользуемые UI-компоненты
│   ├── navigation/     # Описание экранов навигации
│   ├── screens/        # Экраны приложения
│   ├── theme/          # Material 3 тема
│   └── utils/          # Утилиты для цветов и иконок
├── viewmodel/          # ViewModels
├── widget/             # App widget
├── ChronoTrackApp.kt   # Application-класс
└── MainActivity.kt     # Главная Activity
```

## Сборка

```bash
./gradlew assembleDebug
```

Требования: Android SDK 35, JDK 11+, Gradle 8.8+.

## Минимальные требования

- Android 7.0 (API 24)
- Разрешение на уведомления (Android 13+)
